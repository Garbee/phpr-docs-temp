Installation Steps

  1. Prepare a directory on your server that is empty. It can be a sub-directory, domain root or a sub-domain.
  2. Unpack the install archive to the prepared sub-directory, domain root or a sub-domain.
  3. Grant writing permissions for Apache on the installation directory and all its subdirectories and files.
  4. Navigate to the install.php script in your web browser.
  5. Follow the installer instructions.

Notes

If you have installed the script to a subdirectory, you will need to edit the file .htaccess and uncomment the following:

# Uncomment if your site is not located in the root
RewriteBase /subdirectory/
