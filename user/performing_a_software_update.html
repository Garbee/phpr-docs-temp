--- layout: default title: Performing a software update ---

For security reasons it is recommended that you keep your site software as up to date as possible.

How to update software

To update your site, log in to your Administration Area and select Updates from the System menu. From here you can check to see if there are any updates available by clicking Check for updates. A popup window will appear that will show you of any updates you can apply.

If there are available updates you can click Update Software to download and install them. However if there are no updates you can still request the latest version of the script by clicking Force update.

When an update is performed, the following will occur:

  • Core files and modules will be updated
  • The database will be upgraded
  • Your website theme will not be changed.
