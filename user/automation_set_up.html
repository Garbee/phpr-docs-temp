Some features of your script require a cron job to be set up. Most automated features are linked to the cron table, such as:

  • Requests expiring
  • Subscription renewals
  • Deferred processing
  • Internal maintenance house keeping tasks

Set up using cPanel

Click the Cron Jobs link under your cPanel account and choose Advanced (Linux Style). Enter your email address for collecting information about cron execution. Copy the command to be run as cron:

/usr/local/bin/php -q /home/YOUR_USERNAME/public_html/cron.php

Replace YOUR_USERNAME with your cPanel username then schedule the time for this job to execute every 5 minutes.

Apply the changes and click Back to return to the Cron Management page

Set up using Shell

Run the following command

crontab -e

This will open your default text editor, where you will have to insert the following lines of code to be executed:

MAILTO=myemail@mysite.com
*/5 * * * * /usr/local/bin/php -q /home/YOUR_USERNAME/public_html/cron.php

Save the file and exit the editor.
