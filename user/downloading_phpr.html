Getting a copy of PHPR is a striaght forward process. First you must register an account with PHPR and create your first property. A property is a website and you should create 1 property per website.

Using this method has some advantages over a standard installation process.

  • Easily deploy functionality to each website
  • Obtain software updates using 1-click updates
  • Monitor your websites from a single point
