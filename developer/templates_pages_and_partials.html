Building Pages

Building pages is a simple process credited to the system structure and API used in the software. The crux of building pages is the Page Function in addition to the AJAX framework.

  • Page Function
    • - a collection of PHP code either provided or written by you that is executed before a page loads. Generally used to prepare data for being displayed on the page and/or receive data from the user.
  • AJAX Framework
    • - a simple Javascript interface that lets you request or submit information without refreshing the entire page.

In addition to these powerful tools, you are also permitted to use HTML or PHP code inside pages.
