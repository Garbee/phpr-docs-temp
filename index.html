Welcome to PHPR

PHPR is a rapid development framework and CMS for people who build custom web applications using PHP. It is an OOP MVC framework which contains features that you typically find in other frameworks. PHPR also has some unique features like deferred bindings, behaviors and some very helpful Active Record extensions.

Who should use PHPR?

You should consider using PHPR if:

  • You want a simple framework with a small footprint.
  • You want clean, extensible and modular code and data structures.
  • You want a powerful CMS out of the box.
  • You want to apply DRY principles.
  • You prefer convention over configuration.
  • You don't want to be forced to learn a templating language.
